# Best Los Angeles Flowers 🌸

Free, open-source flower tools built by [Bloom Boom Boutique Florist](https://bloom-boom.shop) — a boutique flower shop in West Hollywood serving all of Los Angeles.

No frameworks, no build step, no tracking. Just three genuinely useful static pages you can open in any browser.

## The tools

| Tool | What it does |
|------|--------------|
| [Wedding & Event Flower Calculator](wedding-flower-calculator/index.html) | Estimates stem counts (focal, secondary, filler, greenery) and realistic Los Angeles flower costs for bouquets, boutonnieres, centerpieces, and ceremony pieces. |
| [Flower Care & Vase Life Guide](flower-care-guide/index.html) | Care instructions for 30 popular cut flowers — vase life, trimming and water tips, ethylene sensitivity, and cat/dog safety. |
| [LA Seasonal Flower Guide](la-seasonal-flowers/index.html) | Month-by-month guide to what's fresh, at peak, and California-grown at the Los Angeles flower markets. |

## Running locally

Clone and open `index.html` in a browser — that's it. Everything is plain HTML, CSS, and vanilla JavaScript.

```
git clone https://github.com/justingleader/best-los-angeles-flowers.git
open best-los-angeles-flowers/index.html
```
